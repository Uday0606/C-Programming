# C Programming Practice

This repository contains C programming implementations covering core concepts such as:

- Basic Programming
- Arrays
- Strings
- Pointers
- Structures
- Bit Manipulation
- Searching Algorithms
- Linked Lists (Singly Linked List)


## Purpose
To strengthen fundamentals in C programming with a focus on concepts relevant to embedded and low-level programming.

## How to Run
Compile using GCC:

gcc filename.c  
./a.out
